** Informações do programa **

Linguagem Utilizada: Python 3.9
OpenGL utilizado: PyOpenGL 3.1.5
Biblioteca para criação e configuração da janela: GLUT (é instalado juntamente como PyOpenGL)

Tamanho da Janela: 640x480

Para executar, rode: python3 object_rotation.py

Para rodar, é necessário possuir as bibliotecas citadas acima, elas podem ser encontradas em: https://www.lfd.uci.edu/~gohlke/pythonlibs/

Comandos do Teclado:
    
    * Movimentação do Objeto no eixo X   -> w, s
    * Movimentação do Objeto no eixo Y   -> a, d
    * Movimentação do Objeto no eixo Z   -> q, e
    * Aumentar o tamanho do Objeto       -> +
    * Diminuir o tamanho do Objeto       -> -
    * Alterar o eixo de rotação para o X -> 8, 2
    * Alterar o eixo de rotação para o Y -> 4, 6
