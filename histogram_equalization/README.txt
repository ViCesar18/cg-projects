- As imagens para equalização devem estar na pasta 'images';

- Os histogramas são mostrados na tela conforme a execução do programa;
Obs: sempre que aparecer a janela de um histograma, ela deve ser fechada para o programa continuar a execução;

- A imagem equalizada (resultado) é mostrada na tela, mas também é armazenada na pasta 'equalized-images';

** Importante!! **
Equalizar o histograma de imagens coloridas (como imagens RGB) pode alterar o equilíbrio das cores.
Recomenda-se equalizar imagens com uma só camada (grayscale).

** Execução **
python3 histogram_equalization.py
